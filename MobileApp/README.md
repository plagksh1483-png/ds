## Описание
В этой папке находится мобильное приложение, предназначенное для взаимодействия пользователей с нашим сервисом на мобильных устройствах.

## Компоненты/модули
- Экран входа и регистрации
- Основной интерфейс пользователя
- Модули работы с сетью и API
- Локальное хранение данных
- Настройки и пользовательские параметры

## Взаимодействие с другими частями проекта
Мобильное приложение взаимодействует с серверным API (`ServerApi`) для получения и отправки данных. Также оно использует ресурсы и компоненты из папки `Web` для интеграции с веб-частью, если предусмотрена совместная работа или обмен данными.
