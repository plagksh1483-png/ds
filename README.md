## Краткое описание идеи
Этот проект представляет собой комплексное решение, включающее мобильное приложение, серверный API и веб-сайт, объединенные для предоставления удобного и функционального сервиса для пользователей.

## Структура репозитория
- **MobileApp** — мобильное приложение для iOS и Android, обеспечивающее основной интерфейс взаимодействия пользователей с системой.
- **ServerApi** — серверный интерфейс, реализующий API для обмена данными между мобильным приложением и веб-частью проекта.
- **Web** — веб-сайт, предоставляющий доступ к функционалу через браузер, отображение информации и управление настройками.

Каждая часть проекта взаимодействует друг с другом через API и совместно реализует поставленные бизнес-задачи.
