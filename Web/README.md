## Описание
В этой папке находится веб-сайт — фронтенд часть проекта, предназначенная для взаимодействия с пользователями через браузер.

## Компоненты/модули
- Страницы и шаблоны интерфейса
- Модули работы с API и серверным интерфейсом
- Механизмы авторизации и регистрации
- Взаимодействие с серверным API (`ServerApi`)
- Обработка и отображение данных

## Взаимодействие с другими частями проекта
Веб-сайт обращается к API (`ServerApi`) для получения данных и выполнения операций. Также он может взаимодействовать с мобильным приложением через API или другие обмены данными, если предусмотрено.
